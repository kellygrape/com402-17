Recipe Project

Let's talk about our recipe project. Thinking about what we need to do (print a recipe to the screen, then print another recipe to the screen), what can we do to function-ize our code?

Adding event listeners

document.querySelector(el).addEventListener(e,function(){ // code here });
theElementYouWantToTrigger.addEventListener(theEvent,theStuffYouWantToDo);

Using data attributes to share information from HTML to JS



document
  .querySelector('.nextrecipe')
  .addEventListener('click',function(){
    this.getAttribute('data-recipe'));
    // will return "pancakes" or "cookies", as a string
  });

Save it as a varible (so we can use it!)

document
  .querySelector('.nextrecipe')
  .addEventListener('click',function(){
    var recipetoshow = this.getAttribute('data-recipe'));
    // will return "pancakes" or "cookies", as a string
    
  });

Understanding data types and how to change / transfer data types

document
  .querySelector('.nextrecipe')
  .addEventListener('click',function(){
    var recipetoshow = this.getAttribute('data-recipe'));
    console.log(typeof recipetoshow);
    // should print "String" in the console
  });

Changing things from Strings to Numbers


var myAge = "33"; // this is a string

console.log(typeof myAge);

var myAgeNum = parseInt(myAge);

console.log(typeof myAgeNum);

Changing things from Strings to ... variable names?

Our data attributes are holding our variable names. So ... we need to understand that the string is actually a variable name. We need to get JS to evaluate it.



document
  .querySelector('.nextrecipe')
  .addEventListener('click',function(){
    var recipetoshow = this.getAttribute('data-recipe'));
    console.log(typeof recipetoshow);
    eval(recipetoshow);
    console.log(eval(recipetoshow));  
    // important to know - this doesn't DO anything to the value.
    // it just evaluates it and then waits to see what you want to
    // do with the evaluation.
    // so, we need to save the evaluation result in something else.
    var evaledRecipe = eval(recipetoshow); 
    });
